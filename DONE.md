## Completion Policy (Binary Evidence Required)

An item can move from `TODO.md` to `DONE.md` only when all are true:

1. Required checks for that item/phase were run.
2. Evidence is recorded (command output snippet, test report, CI/job link, or reviewer note).
3. Related docs were updated if scope/behavior changed.
4. Reviewer approval is present.

If evidence is missing, the item stays in `TODO.md`.
